# how-many-stars

[![Version](https://img.shields.io/badge/version-0.1.0--alpha-blue.svg)](CHANGELOG.md)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

Live: **https://0xsalt.github.io/how-many-stars/**

Web dashboard showing GitHub star counts for all public 0xsalt repositories, sorted by popularity.

## Usage

```bash
# Start development server with hot reload
bun run dev

# Start production server
bun run start
```

Open http://localhost:3000 to view the dashboard.

## Features

- Displays all public repositories from 0xsalt
- Shows star count for each repository
- Sorted by stars (highest first)
- Shows total star count
- Links to each repository on GitHub
